①課題3-1を以下の情報を表示するようにプログラムを拡張しなさい。
1.トップレベルドメイン
2.ホスト名
3.パス
4.ファイル名(拡張子も含む)

        
                from urllib.parse import urlparse
                print("URLを入力してください")
                url = input()
                
                parsed_url = urlparse(url)
                print("プロトコル : "+parsed_url.scheme)
                netLoc = parsed_url.netloc
                splitedNetLoc = netLoc.split(".")
                print("トップレベルドメイン : ."+splitedNetLoc[3])
                print("ホスト名 : "+splitedNetLoc[0])
                path = parsed_url.path
                splitedPath = path.split("/")
                splitedPath.pop(-1)
                joinedPath = "/".join(splitedPath)
                print("パス : "+joinedPath)
                splitedPath = path.split("/")
                print("ファイル名 : "+splitedPath.pop(-1))

課題4

下記のプログラムは、正の整数xを二乗した値を出力するプログラムとなっている。xが負の値の場合にも最後の出力結果が正しい表示となるように修正しなさい。

        
                x = -3
                ans = 0
                itersLeft = x
                if x < 0:
                    while (itersLeft != 0):
                        ans = ans + x
                        itersLeft = itersLeft + 1
                    print(str(x) + ' * ' + str(x) + ' = ' + str(-ans))
                elif x > 0:
                    while (itersLeft != 0):
                        ans = ans + x
                        itersLeft = itersLeft - 1
                    print(str(x) + ' * ' + str(x) + ' = ' + str(ans))
                else:
                    print('0 * 0 = 0')
