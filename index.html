課題2 解説


#1-1 「print()」 データを出力する関数
#使用例
print("こんにちは")

コンソール結果:　
こんにちは



#1-2 「input()」 ユーザーの入力した値を「str」型で格納する関数
#使用例
a = input()
print(a)

コンソール結果:
#テキストボックスが表示される⇨例えば「15」と入力する
15




#1-3 「int()」 int型(数値型)に変換する関数
#使用例
a = input() #例えば、3と入力する

b = input() #例えば、2と入力する

answer = a + b
#どう出力されるでしょうか....?
答えは、「エラー」
input()関数はstr型(文字列型)のため「+」や「-」のような算術演算子を使うことはできない。
そこで、計算が可能な値(int型 または 数値型)に変えてあげる必要がある!!


a = input() #3を入力したとする
b = input() #3を入力したとする

a = int(a) #aがstrで格納されていたためint型に直す
b = int(b) #bがstrで格納されていたためint型に直す

print(a+b)

コンソール結果:
5




#1-4 「if-else(条件分岐)」
if 条件1:
    条件1を満たしたときに行う処理
else:
    条件1を満たしていない時に行う処理

#使用例
a = 10

if a > 5:     # aが5より大きいかを判別
    print("aは5より大きな数値です")  #条件を満たしていたのでこちら側の処理が行われる。
else:
    print("aは5以下の数値です")

コンソール結果:
aは5より大きな数値です

問題3-2 解答


from urllib.parse import urlparse
print("URLを入力してください")
url = input()

parsed_url = urlparse(url)
print("プロトコル : "+parsed_url.scheme)
netLoc = parsed_url.netloc
splitedNetLoc = netLoc.split(".")
print("トップレベルドメイン : ."+splitedNetLoc[3])
print("ホスト名 : "+splitedNetLoc[0])
path = parsed_url.path
splitedPath = path.split("/")
splitedPath.pop(-1)
joinedPath = "/".join(splitedPath)
print("パス : "+joinedPath)
splitedPath = path.split("/")
print("ファイル名 : "+splitedPath.pop(-1))

下記のプログラムは、正の整数xを二乗した値を出力するプログラムとなっている。
xが負の値の場合にも最後の出力結果が正しい表示となるように修正しなさい。

x = -3
ans = 0
itersLeft = x
    while (itersLeft != 0):

    print(str(x) + ' * ' + str(x) + ' = ' + str(-ans))

問題4 解答


x = -3
ans = 0
itersLeft = x
if x < 0:
    while (itersLeft != 0):
        ans = ans + x
        itersLeft = itersLeft + 1
    print(str(x) + ' * ' + str(x) + ' = ' + str(-ans))
elif x > 0:
    while (itersLeft != 0):
        ans = ans + x
        itersLeft = itersLeft - 1
    print(str(x) + ' * ' + str(x) + ' = ' + str(ans))
else:
    print('0 * 0 = 0')

問題4 解説


#かけ算を加法で表そうとする問題。(3×3を3+3+3みたいな)
#問題の例のままだと、xがマイナスや０の時エラーが起きてしまう....
#その原因はxの個数分加算させる「itersLeft」がマイナスの時は0に収束してくれないから。
#例えば、x=3(プラス)の時はitersLeftの初期値は3、whileで繰り返されていくうちに3,2,1,0と減っていき、
#whileの条件の(itersLeft != 0)から外れる(itersLeftが0でない間繰り返される)
#しかし、x=3(マイナス)の時はitersLeftの初期値は-3、whileで繰り返されていくうちに-3,-4,-5,-6.....と減っていくが、
#whileの条件の(itersLeft != 0)から外れないので、「初期条件」で分岐させてあげる必要がある。
#今回は、+,-,0の3つに分岐させる。



#4 「(if-elif-else)」
#基本的に考え方はif-elseと同じ。
if 条件1:
    条件1を満たしたときに行う処理
elif 条件2:
    条件2を満たしたときに行う処理
else:
    上記に何も当てはまらなかった場合に行う処理

#if-elif-elif-.......elif-elseみたいに複数に分岐させることも



x = -3
ans = 0
itersLeft = x
if x < 0:
    while (itersLeft != 0):
        ans = ans + x
        itersLeft = itersLeft + 1
    print(str(x) + ' * ' + str(x) + ' = ' + str(-ans))
elif x > 0:
    while (itersLeft != 0):
        ans = ans + x
        itersLeft = itersLeft - 1
    print(str(x) + ' * ' + str(x) + ' = ' + str(ans))
else:
    print('0 * 0 = 0')
